# Vault-OS Next.js Scaffold

This directory contains a Next.js App Router foundation for the Vault-OS cinematic operating-system experience.

## Included foundations
- Multi-user role baseline with hidden-owner role placeholders.
- Mandatory 2FA login API contract scaffold.
- Overshell rendering layer with ambient pixel-entity placeholder.
- Glassmorphism visual system starter styles.
- API route placeholders for auth and audit streams.
